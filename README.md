# GGV_spring 팀 프로젝트

## 프로젝트 소개

CGV를 참고하여 만든 영화 예매 사이트 입니다.

제작년도 : 2019

## 개발 환경
- Java 8 
- jdk 1.8
- IDE : STS3, Eclipse, VS code
- Database : Oracle DB(11g)
- ORM : Mybatis
- Server : aws , Apache
